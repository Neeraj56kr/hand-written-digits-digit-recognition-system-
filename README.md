# 🧠 hand written digits (digit recognition system) Neural Digit Vision

A modern handwritten digit recognition web application powered by Deep Learning and Flask.

This project uses an Artificial Neural Network (ANN) trained on the MNIST dataset to predict handwritten digits with high accuracy.

---

## 🚀 Features

- ✍️ Draw digits directly on canvas
- 📤 Upload handwritten digit images
- 🤖 Real-time digit prediction
- 📊 Confidence score display
- 🎨 Modern premium UI
- ⚡ Fast Flask backend
- 🧠 TensorFlow/Keras ANN model

---

## 🛠️ Tech Stack

- Python
- Flask
- TensorFlow / Keras
- OpenCV
- NumPy
- HTML
- CSS
- JavaScript

---

## 📂 Dataset

MNIST Dataset  
- 60,000 training images
- 10,000 testing images
- 28×28 grayscale handwritten digits

---

- Dense Hidden Layers
